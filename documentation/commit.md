# Commits
### How commits must be done in this repo:

`[type] - [description]`

`[What affects]`

`[Trello activity link] ->` [link](https://trello.com/laboratorio984)

#### List of types:

- New Feature
- Change
- Breaking change (for functionalities major changes)
- Fixed


#### List of 'what affetcs':
- Style
- Back-end
- Database
- Front-end
- Documentation
